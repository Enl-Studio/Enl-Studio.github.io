新成员申请

加入 Enl Stduio 并不是一件困难的事。为了您能更加顺利地加入本组，请认真阅读以下注意事项。

在提交申请前，请检查你是否满足以下两点：

  • 现在（曾经）是和我们（任一现有组员）同一学校的学生
  • 与任一现有组员相识

申请流程：

编写一份简单的个人身份说明，至少包含以下部分：

  • 所在学校及院系
  • 国内社交平台的 ID（方便在某协会群内喊人）
  • Telegram 账户 ID

可以选择加入以下信息方便更好地玩耍（没列出的你想写也可以）：

  • 国外社交平台账户 ID（方便大家互 Fo）
  • 国内邮箱账户
  • GPG PubKey
  • OSU ID（Click on circles!）
    …

提交申请的方式：

  • 由你认识的成员发送到成员相关群组内
  • GPG 加密投递到任意你认识的群组成员邮箱内
    （透过此方法投递必须注明认识成员的常用 ID 或其他可辨识的身份识别信息）

申请审核：

原则上已有在线成员多数同意后即接受申请

申请完成：

申请成功后，将由已有成员或管理员拉入相应群组。

加入群组后需要注意的：

  • 出于某些考量，长时间未上线且失去安全联络手段的账户会被移除
  • 群组内提供的服务没有硬性限制滥用，任何有意的滥用行为都将直接拉黑
  • 群组内消息在没有收到发送人的允许前，不允许以任何形式转发

然后就可以愉快玩耍了~
